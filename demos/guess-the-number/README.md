# Guess the Number

An interactive browser-based number guessing game built with HTML, CSS, and JavaScript. Players attempt to guess the correct number while receiving instant feedback and tracking the number of attempts required to solve the challenge.

---

## Features

- Interactive number guessing gameplay
- Input validation for valid guesses
- Real-time feedback after each attempt
- Attempt counter
- Reset functionality to start a new game
- Responsive design for desktop and mobile devices

---

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
