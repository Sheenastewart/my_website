# Flashcard Quiz

An interactive flashcard application built with HTML, CSS, and JavaScript that helps users study software engineering concepts through animated flashcards, intuitive navigation, and a responsive user interface.

The application demonstrates core front-end development concepts including DOM manipulation, event-driven programming, responsive design, and CSS animations while providing an engaging learning experience.

---

## Features

- Interactive flashcard flip animation
- Next and Previous card navigation
- Dynamic question and answer loading
- Progress indicator displaying current card number
- Responsive design for desktop and mobile devices

---

## Technologies Used

- HTML5
- CSS3
- JavaScript
