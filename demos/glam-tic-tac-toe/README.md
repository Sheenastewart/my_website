# Glam Tic-Tac-Toe

A modern browser-based Tic-Tac-Toe game built with HTML, CSS, and JavaScript. The application features both single-player and two-player gameplay, live score tracking, responsive design, and an intelligent computer opponent that can recognize winning opportunities and block the player.

---

## Features

- Two-player gameplay
- Single-player mode against a computer opponent
- Computer AI with offensive and defensive move logic
- Live score tracking
- Win and draw detection
- Winning combination highlighting
- Reset game functionality
- Reset scoreboard functionality
- Responsive design for desktop and mobile devices

---

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
