# Resume Website

A professional software engineering portfolio website built with HTML, CSS, and JavaScript. The website serves as an online resume, highlighting my technical skills, professional experience, certifications, featured projects, and contact information in a clean, responsive interface.

---

## Features

- Responsive design for desktop and mobile devices
- Professional landing page with engineering-focused branding
- About Me section highlighting technical background
- Technical Skills section organized by category
- Featured Projects section linking to GitHub repositories
- Resume page with downloadable PDF resume
- Contact section with LinkedIn and email information
- Clean navigation with smooth scrolling

---

## Technologies Used

- HTML5
- CSS3
- JavaScript

---

## What I Learned

This project strengthened my understanding of responsive web design, semantic HTML, modern CSS layout techniques, and JavaScript-driven user interaction. It also reinforced the importance of creating software that is both technically functional and professionally presented.
