# Software Engineering Portfolio

A professional portfolio and online resume for Sheena Stewart, a Software Engineer and AWS Certified Cloud Practitioner with more than 10 years of technical experience.

## Features

- Professional software engineering introduction
- Technical skills overview
- Featured front-end projects
- Downloadable PDF resume
- Online resume page
- Contact information
- Responsive design

## Technologies Used

- HTML5
- CSS3
- JavaScript

## Project Structure

- `index.html` — Main portfolio homepage
- `resume.html` — Online resume page with PDF download
- `style.css` — Site-wide styles
- `main.js` — Smooth scrolling and form handling
- `assets/resume/` — Resume PDF storage
